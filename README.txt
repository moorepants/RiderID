Rider Identification Software in Matlab

This is the accompanying software to derive the results in the paper:

Rider control identification in bicycling using lateral force pertubation
tests. A.L. Schwab, P.D.L Lange, R.Happee, and Jason K. Moore. Multibody System
Dynamics, 2013

Software Requirements:
- Matlab recommended version 2011a or higher
- Matlab Optimization toolbox
- Matlab Control System toolbox
- Matlab DSP System toolbox

Installation:
- unpack RiderID folder to any location
- set the RiderID as the project folder in Matlab.
- Run main.m
